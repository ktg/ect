The original dataspace example application, showing use of the low-level (non-beans, cross-language) API and custom IDL'd data types (rather than tuples).

Starting points:

  • {@link equip.data} The core dataspace package
  • {@link equip.data.beans.DataspaceBeanTupleTest} A version using the simpler {@link equip.data.beans.IDataspace} API and generic tuple data items ({@link equip.data.Tuple}).
  • {@link equip.data.beans.DataspaceBeanEventTest} A version using the simpler {@link equip.data.beans.IDataspace} API and generic tuple events ({@link equip.data.TupleEvent}) rather than data items.
  • {@link equip.data.TupleTest} A version using the same low-level API but with generic tuple data items ({@link equip.data.Tuple}) rather than custom IDL'd data types.
  • {@link equip.data.beans.SimpleBeanTest} A version using the beans only (no dataspace API programming).
