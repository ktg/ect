Core types for EQUIP and EQIDL, including the top class ({@link equip.runtime.ValueBase}), and serialisation, plus various C++ IDL helpers such as object factories, smart pointers and sequences.

Starting point

  • {@link equip.runtime.ValueBase}, the top class, including pattern matching and serialisation support.
  • {@link equip.runtime.SingletonManager}, for getting singleton instances of classes.
  • {@link equip.runtime.ObjectInputStream} and {@link equip.runtime.ObjectOutputStream}, the EQUIP stream classes.
