Equip is (mainly) a data-space based coordination platform, which
combines ideas from Tuple spaces with improved performance for 
replicated publish-subscribe pattern-based interaction. Equip was 
developed for research by the UK EQUATOR IRC (http://www.equator.ac.uk).

Equip4j is a Java version of Equip.
