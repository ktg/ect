Introduction

There are two mailing lists that you can subscribe to if you are interested in ECT. These mailing lists are hosted by SourceForge, and are also used by individual who are interested in EQUIP4J section of this website. The mailing lists that are available are:
  • the equip-users mailing list - intended for ECT or EQUIP4J users to post comments, requests for help, requests for features and bug reports, and as a mechanism for announcements about software to be made
  • the equip-developers mailing list - intended to provide a forum for ECT or EQUIP4J developers to discuss issues relating to development
Both of these lists are currently very low traffic (~1 message a month).

Using the mailing lists

To subscribe to either mailing list, go to the EQUIP project mailing list page, which is hosted by SourceForge, and which also provides facilities to browse the mailing list archives, and to unsubscribe and modify your mailing list preferences.
