Notes on ECT AR Authoting Components

Author: Alastair Hampshire
Date: 3/1/07

Available Components

There are 3 parts to the AR authoring components within the equip.ect.components.arauthoring package:
  • scenegraph (3d output component)
  • transformation (a set of transformation matrix related components e.g. creating a rotation matrix from 3 scalars)
  • audioplayer (a very simple .wav player)
The above components are intended to be used in combination with the following components from the equip.ect.components.visionframework package:
  • capture (JMF video capture)
  • artoolkit (AR marker tracking)

Building

The AR authoring components are not in the default build, so build with:
  • ant arauthoring
To build the visionframework components call:
  • ant visionframework

Resources

The following resources can be found in the <ect install>/ect/java/resources/arauthoring/ directory:
  • 3dmodels (some models that can be loaded with the scene graph component
  • audio-clips (some -currently only 1-  audio clips that can be played with the audio player component)
  • patterns (a doc containing the pattern file urls for the AR marker tracking component and a number of pdfs of AR markers that can be printed)
